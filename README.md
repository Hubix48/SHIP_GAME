# Gra Statki
Projekt gry w statki wykonany w technologii JavaFX.

## Funkcje:
- tryb lokalny (2 graczy)
- gra z botem (3 poziomy trudności)
- multiplayer TCP (klient + serwer)
- boosty specjalne: RADAR, SWAP, TROLL
- eksport historii rozgrywki do pliku

## Uruchamianie:

1. Kompilacja projektu (tworzony jest folder gry):

   build.bat

2. Uruchomienie serwera:

   run_server.bat

3. Uruchomienie klienta:

   run_client.bat

4. Uruchomienie gry:

   run.bat

## Wymagania:

- Java 21 (JDK)
- JavaFX
